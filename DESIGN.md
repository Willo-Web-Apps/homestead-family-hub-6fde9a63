# Design DNA — Hearthstead

## Visual Personality: organic-natural
## Layout: storytelling
## Typography: elegant-serif

### Color Palette
- Primary: #3D5A2E — deep forest green → `oklch(0.36 0.08 140)`
- Accent: #C4773B — harvest amber/terracotta → `oklch(0.60 0.12 55)`
- Surface: #F7F1E8 — warm cream/parchment → `oklch(0.96 0.015 85)`
- Text: #2A2017 — rich dark bark → `oklch(0.18 0.025 75)`

### Font Pairing
- Display: Cormorant Garamond — weights 400, 500, 600, 700 (luxury editorial serif, humanist warmth)
- Body: Lora — weights 400, 500 (readable elegance, warm editorial feel)

### Unique Design Choices
1. **Full-bleed hero with layered OKLCH gradient overlay** — deep forest green + harvest amber orbs over cream, with subtle noise grain texture, creating a golden-hour farmhouse atmosphere.
2. **Organic flowing section dividers** — SVG wave shapes between sections, giving the layout a handcrafted, hand-tended feel vs. clean geometric cuts.
3. **Typography as primary visual element** — dramatically oversized Cormorant Garamond headlines with generous letterspacing and varied weights create an editorial magazine feel that no competitor has.
4. **Three-step journey cards with illustrated numeral accents** — large, faded numerals in forest green behind each step card create depth and visual rhythm.
